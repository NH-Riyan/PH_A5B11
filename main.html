What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML : We can get or set the HTML content inside an element, including all HTML tags and text by using innerHTML.
2.innerText : By using innerText we can get or set the visible text content of an element
3.textContent : By using textContent we get or set all text content, including hidden elements.

What is event delegation in the DOM?

Event delegation is a way to handle events by adding a single event listener to a parent element instead of adding listeners to multiple child elements.An event happens on a child,the event bubbles up to the parent.

What is event bubbling in the DOM?

Event bubbling is when an event starts on the child element and moves up through its parent elements until it reaches the top of the DOM.
